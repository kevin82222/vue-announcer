# Installation

## Package

#### NPM
```shell
npm install -S vue-announcer
```

#### Yarn
```shell
yarn add vue-announcer
```

## Basic usage

```javascript
import Vue from 'vue'
import VueAnnouncer from 'vue-announcer'

Vue.use(VueAnnouncer)
```

In your `App.vue`
```vue
<template>
  <div>
    <vue-announcer /> <!-- You can place it anywhere in your application -->
    ...
  </div>
</template>
```
